Product Modules

功能设计

项目总览

展示简历、面试、JD 匹配和投递数据；根据真实记录计算求职准备度，自动给出短板和下一步行动。

简历实验室

支持录入、上传、编辑、原件替换、PDF/Word 导出转换、客观分析、JD 优化和技能图谱。

面试训练场

完整模拟真实面试流程，包含专业知识面试、题库练习、语音输入、真实录音回放和 AI 表达分析。

投递看板

记录公司、岗位、阶段、城市和备注，支持阶段推进、跟进话术生成、风险提示和薪资评估。

AI 求职教练

读取当前简历、投递、面试训练上下文，生成可执行建议和求职作战报告。

多模型网关

支持 GLM、DeepSeek、Kimi / Moonshot，可配置 API Key、自定义模型 ID，并提供本地规则兜底。

Career Loop

业务闭环

1. 建立简历资产上传或录入简历，保存原件和解析文本。
2. 复制真实 JD从招聘平台复制岗位描述，提取关键词和风险点。
3. 定制简历生成匹配分、差距清单、项目改写和面试讲述点。
4. 模拟面试按真实流程训练，并记录回答、评分和语音复盘。
5. 投递跟进维护机会池，推进阶段，生成跟进话术和求职报告。

Agent Ability

智能体能力设计

  • 通过多模型网关统一 GLM、DeepSeek、Kimi 的调用参数和返回结构。
  • 每个 AI 功能都有本地兜底逻辑，避免 API 不可用时系统完全失效。
  • 面试训练保留 stage 状态，能按流程推进问题，而不是普通聊天。
  • 语音分析结合文本和声音指标，给出语速、停顿、爆音、结构感建议。
  • 训练记录和投递记录沉淀为 AI 教练上下文，支持持续复盘。

Resume Description

可写进简历的项目描述

项目名称：基于 AI 智能体的求职辅助 Web 系统

设计并实现面向求职场景的 AI Web 系统，覆盖简历管理、JD 匹配、模拟面试、语音表达分析、投递追踪和 AI 求职教练等模块。封装多模型 AI 网关，支持 GLM、DeepSeek、Kimi / Moonshot 切换，并设计本地规则兜底，保证 API 不可用时核心流程仍可演示和测试。实现 JD 驱动的简历优化能力，输出关键词命中、能力差距、匹配分、项目经历改写和面试讲述要点。使用 Flask + SQLite 提供后端接口，原生 JavaScript + Chart.js 构建前端交互，并补充核心接口测试，形成可演示、可测试、可迭代的实训项目。
